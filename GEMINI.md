# GEMINI.md

## Project Context
Educational Node.js/Express web application for CESMAC Information Systems students (3rd period). Uses EJS for templating and MongoDB for data storage.

## Standards
- Vanilla JavaScript (CommonJS `require` syntax).
- Express for routing and server setup.
- EJS for server-side view rendering.

## Constraints
- Keep code simple and accessible for students; do not over-engineer.
- Do not use TypeScript or single-page application frameworks (React, Next.js, etc.).
- Maintain clean separation between server logic (`index.js`), views (`views/`), and static assets (`public/`).
